Products Search REST API On Firebase Cloud

Product Search API endpoint returns a list of products information and is also capable of providing different search and filter conditions applied to filter down the search restuls. These filter conditions are passed as query parameters. Below are allowed query parameters for the endpoint.


  • search
  • minPrice
  • maxPrice
  • minReviewRating
  • maxReviewRating
  • minReviewCount
  • maxReviewCount
  • inStock
URL:  https://mobile-tha-server-8ba57.firebaseapp.com/walmartproducts/{pageNumber}/{pageSize}?query_parameters

Examples:


  • https://mobile-tha-server-8ba57.firebaseapp.com/walmartproducts/1/3?search=mount&inStock=true
  • https://mobile-tha-server-8ba57.firebaseapp.com/walmartproducts/2/5?inStock=true&minReviewRating=4
  • https://mobile-tha-server-8ba57.firebaseapp.com/walmartproducts/1/10?minPrice=500&maxPrice=2000&inStock=true&minReviewCount=2

Product Search REST API Tech Stack
  • Node JS and Express
  • MongoDB Atlas Cloud
  • Firebase Cloud - Hosting & Storage
